# SpSk Marketplace

Polished AI agent skills for Claude Code.

## Install

```bash
# Add the marketplace
claude plugins marketplace add spsk-dev/marketplace

# Install any plugin
claude plugins install tasteful-design@spsk
claude plugins install code-review@spsk
claude plugins install consensus@spsk
```

## Plugins

| Plugin | What it does |
|--------|-------------|
| **tasteful-design** | 7-specialist design review + SPA flow audit |
| **code-review** | 7-agent multi-model PR review |
| **consensus** | 3-model conclusion validation (1-10 confidence) |
